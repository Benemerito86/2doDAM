# Introducción a Hibernate

## 1º PARTE: 
### Se pide implementar en java con tecnología Hibernate el siguiente enunciado:

- Implementar un refugio de animales
- Cada animal tiene un nombre, especie, edad, estado y una descripción de cómo se perdió
- Tendremos que implementar funciones con el objetivo de:
- Registrar nuevos animales que lleguen a mi refugio
- Buscar animales por especie
- Actualizar el estado en el que se encuentran (recién abandonado, tiempo en el refugio, próximamente en acogida)


## 2ºPARTE: 
Se amplía los casos donde tenemos que modelar una persona que es dueña de varios animales guardando DNI, nombre, email y que los animales pueden tener varias clasificaciones ("mamífero y carnívoro") identificandose con un código y el nombre de la clasificación
