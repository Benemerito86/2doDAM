## Crear un nuevo proyecto JAVA
### Preparar un programa que ejecute varios comandos (al menos 6 comandos):
- Notepad, calc, shel, etc. Uno detrás de otro, de forma secuencial y hacer que el programa obtenga el código de finalización de cada uno de ellos.
- Para cada programa imprimir el nombre y su código de finalización.
- Se debe probar a poner aplicaciones que no existan o comandos con parámetros incorrectos.
